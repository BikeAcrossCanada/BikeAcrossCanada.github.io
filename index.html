Trans Canada Bike Route Network

This is the proposed National Cycling Route Network for Canada. Below is the Google Map (link to see it on the web and/or Google Maps smartphone app). It may take an actual moment for the internet elves to load the map below.

Download KMZ file for use with the free Organic Maps Smartphone App.


(Left panel) The free desktop (windows/Mac) software 'Garmin BaseCamp' is used to manage the primary database. You can download the software from the Garmin website, garmin.com/en-CA/software/basecamp/

You can download the full GPX file containing the network and all of the subfolders with lots of POI's (points of interest) and other useful KMZ and GPX files. https://archive.org/download/BikeAcrossCanadaRouteNetworkGPSTracksPOIs/Bike%20Across%20Canada%20Route%20Network%20W-E%20%28proposed%29%20Connected%20daily%20OSM%20data%20segments.gpx

This is the GPX file for the network on it's own (without any POIs)



Garmin BaseCamp comes with a very basic map, for the detailed map you can install the free OpenMapChest Canada openmapchest.org/maps/canada/

An older copy of the installation file for 'OpenMapChest Canada' is available ) here (hosted on Internet Archive - Archive.org



(Centre panel) The 'Trans Canada Bike Route /Route Cyclable Transcanadienne - Map 1of4 - 2024 Proposed National Cycling Route Network for Canada & Accommodations POIs' is the primary online map for this project (everything is copied/imported from the Garmin BaseCamp database to the 'Google My Maps' map) This map can be viewed on any device connected to the internet from any web browser. (The Google Maps app is the most commonly used app for Android). https://goo.gl/maps/FRp4kfMjAPACqbgm6

(Right panel) The free Organic Maps offline maps (available in all flavours from organicmaps.app) is the best app to view the network. You can download the KMZ file for just the linear network here (it's a copy of the Google Map without the POIs) https://archive.org/download/BikeAcrossCanadaRouteNetworkGPSTracksPOIs/Trans%20Canada%20Bike%20Route.kmz

You can download the ZIP file which contains all of the POIs and many other KMZ files (such as the The Great Lakes Waterfront Trail in Ontario and the Route Verte Network in Quebec) https://archive.org/download/BikeAcrossCanadaRouteNetworkGPSTracksPOIs/Bike%20Across%20Canada%20Route%20Network%20KMZ%20-%20extracts%20from%20GPX%20file%20ZIP%20file.zip

If your having difficulty with the Zip file, the free Android App 'Trans Canada Bike Route App' has been made to guide you set-by-step to 'unzip' the file and load it onto Organic Maps https://play.google.com/store/apps/details?id=com.wBikeAcrossCanadaRouteNetworkappv12_16984567

And if you have any questions, please ask.
